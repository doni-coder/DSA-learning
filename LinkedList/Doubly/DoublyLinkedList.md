## Doubly Linked List (DLL) — Revision Notes

### What is a Doubly Linked List?
→ A **Doubly Linked List (DLL)** is a linear data structure where each node contains:
- **Previous pointer (prev)**
- **Data**
- **Next pointer (next)**

→ Traversal is possible in **both directions** (forward & backward).  
→ Memory is **non-contiguous**.

---

## Node Structure
→ Each node has three parts:

```text
[prev | data | next]
